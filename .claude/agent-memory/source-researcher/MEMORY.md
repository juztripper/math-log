## Padrões Descobertos

### Conceitos Sempre Presentes em Dados Bivariados
1. Diagrama de dispersão / Nuvem de pontos
2. Coeficiente correlação linear (r ∈ [-1,1])
3. Correlação positiva/negativa/nula
4. Reta de regressão y = ax + b
5. Centro de gravidade
6. Variáveis independente/dependente

### Lacunas nas Fontes
- Fórmulas matemáticas explícitas (fontes recomendam calculadora)
- Dedução método dos mínimos quadrados
