## Terminologia Correta (das fontes)

- **População**: "conjunto de todos os indivíduos (objetos ou pessoas) que possuem pelo menos uma característica comum"
- **Amostra**: "subconjunto da população, que se espera ser representativo dessa mesma população"
- **Variável**: "atributo ou característica que pode assumir diferentes valores numéricos ou não"
- **Análise exploratória**: "fase inicial do tratamento de dados que consiste em organizar e resumir os dados com o objetivo de os descrever"
